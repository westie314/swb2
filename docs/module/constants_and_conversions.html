@file Contains a single module, @ref constants_and_conversions, which contains useful physical constants and basic conversion utilities This module contains physical constants and convenience functions aimed at performing unit conversions. The functions included in this module perform conversions between various temperature and distance units.

establish generic interfaces to single and double precision functions


Uses

    • iso_c_binding
