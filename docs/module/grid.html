@file Contains a single module, grid, which provides support for gridded ASCII data file and data structure operations @ingroup grid Provides support for input and output of gridded ASCII data, as well as for creation and destruction of grid data structures (defined types).

interface to C code that provides a simplified entry point to PROJ4 capabilities: it has been modified so that all C pointers are kept within the C code; no pointers are returned to fortran
